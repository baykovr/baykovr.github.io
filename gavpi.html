GAVPI

gavpi is an open source alternative to voice attack.

Summary

Specifically gavpi is made with StarCitizen in mind, although you could use it for any program.

Essentially a wrapper for windows speech recognition, gavpi is capable of automating complex key press sequences through the use of voice commands.

Direct input keypress emulation is coupled with speech recognition it serves as an excellent addition to many video games or applications.

Please bear in mind that gavpi is under heavy development by only one person, so if you find any bugs or have any comments let me know.

While you're here try out the latest build or check the news page.
